# 二、功能一：代码块复制功能（重点体验）

---

## 2.1 功能说明

在博客详情页：

👉 每个代码块右上角显示【复制】按钮
👉 点击后自动复制代码内容

---

## 2.2 技术方案

👉 前端处理（Markdown 渲染后增强）

---

## 2.3 实现方式（核心）

你现在用 Markdown（Vditor），代码块最终会变成：

```html
<pre><code>...</code></pre>
```

---

## 2.4 Vue实现（DOM增强）

```js
import { onMounted } from "vue"

onMounted(() => {
  document.querySelectorAll("pre").forEach(block => {
    const btn = document.createElement("button")
    btn.innerText = "复制"
    btn.className = "copy-btn"

    btn.onclick = () => {
      const code = block.innerText
      navigator.clipboard.writeText(code)
    }

    block.style.position = "relative"
    block.appendChild(btn)
  })
})
```

---

## 2.5 样式

```css
.copy-btn {
  position: absolute;
  top: 5px;
  right: 5px;
  background: #FF8FB1;
  color: white;
  border: none;
  padding: 5px 10px;
  border-radius: 5px;
  cursor: pointer;
}
```

---

## 2.6 优化（建议）

* 点击后提示：“已复制”
* 防重复插入按钮（注意组件更新）

# 三、功能二：分页系统（必须规范）

---

## 3.1 功能说明

* 每页：10篇
* 支持：

  * 下一页
  * 上一页
  * 输入页码跳转

---

## 3.2 后端实现

---

### 接口

```http
GET /api/blog/page?page=1&size=10
```

---

### Service

```java
public Page<Blog> getBlogs(int page, int size) {
    PageHelper.startPage(page, size);
    return blogMapper.selectAll();
}
```

---

### 返回结构

```json
{
  "list": [...],
  "total": 100,
  "page": 1,
  "pages": 10
}
```

---

## 3.3 前端实现

---

### Vue分页组件

```vue
<div class="pagination">
  <button @click="prev">上一页</button>

  <span>{{ page }}/{{ pages }}</span>

  <button @click="next">下一页</button>

  <input v-model="jumpPage" placeholder="页码" />
  <button @click="jump">跳转</button>
</div>
```

---

### JS逻辑

```js
function next() {
  if (page.value < pages.value) {
    page.value++
    loadData()
  }
}

function prev() {
  if (page.value > 1) {
    page.value--
    loadData()
  }
}

function jump() {
  page.value = jumpPage.value
  loadData()
}
```

# 五、功能四：管理员系统（重点）

---

## 5.1 用户角色设计

---

### user 表新增字段

```sql
ALTER TABLE user ADD COLUMN role VARCHAR(20) DEFAULT 'user';
```

---

## 角色定义：

| 角色    | 权限   |
| ----- | ---- |
| user  | 普通用户 |
| admin | 管理员  |

---

## 5.2 权限能力

---

### 管理员可以：

1. 删除任意博客
2. 删除任意评论
3. 封禁用户（可选）
4. 置顶文章（推荐加）

---

## 5.3 后端权限控制（Spring Security）

---

### 判断管理员

```java
public boolean isAdmin(User user) {
    return "admin".equals(user.getRole());
}
```

---

---

## 5.4 删除博客接口（增强）

```java
public void deleteBlog(Long blogId, User user) {
    Blog blog = blogMapper.selectById(blogId);

    if (!blog.getUserId().equals(user.getId()) && !isAdmin(user)) {
        throw new RuntimeException("无权限删除");
    }

    blogMapper.deleteById(blogId);
}
```

---

---

## 5.5 删除评论接口

```java
public void deleteComment(Long id, User user) {
    Comment c = commentMapper.selectById(id);

    if (!c.getUserId().equals(user.getId()) && !isAdmin(user)) {
        throw new RuntimeException("无权限");
    }

    commentMapper.deleteById(id);
}
```

---

---

## 5.6 封禁用户（推荐）

---

### user表新增字段

```sql
ALTER TABLE user ADD COLUMN status INT DEFAULT 1;
```

---

| 状态 | 含义 |
| -- | -- |
| 1  | 正常 |
| 0  | 封禁 |

---

---

### 登录校验

```java
if (user.getStatus() == 0) {
    throw new RuntimeException("账号已被封禁");
}
```
