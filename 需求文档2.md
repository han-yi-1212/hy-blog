# 二、功能一：点赞数量展示（增强版）

## 2.1 功能说明

在以下位置显示点赞数：

* 博客卡片列表
* 博客详情页

同时：

* 已点赞用户显示“高亮状态”
* 点赞数实时变化（弱实时即可）

---

## 2.2 后端设计（关键）

### 数据结构（Redis）

```bash
key: blog:like:{blogId}
type: SET
value: userId
```

### 点赞数获取

```java
Long likeCount = redisTemplate.opsForSet().size("blog:like:" + blogId);
```

---

### 接口扩展

#### 获取博客列表（增强）

```http
GET /api/blog/list
```

返回：

```json
{
  "id": 1,
  "title": "...",
  "likeCount": 12,
  "isLiked": true
}
```

---

#### 点赞接口返回优化

```json
{
  "likeCount": 13,
  "isLiked": true
}
```

---

## 2.3 前端实现

### Vue 示例

```vue
<div class="like-box" @click="toggleLike(blog.id)">
  <span :class="{ liked: blog.isLiked }">❤️</span>
  <span>{{ blog.likeCount }}</span>
</div>
```

---

### 样式建议

```css
.like-box {
  cursor: pointer;
  color: #aaa;
}

.like-box .liked {
  color: #6CA6FF;
}
```

---

## 2.4 优化建议（面试加分）

* 点赞写 Redis，定时同步 MySQL
* 防重复点赞（SET天然支持）

## 3.2 搜索功能

### 后端接口

```http
GET /api/blog/search?keyword=xxx
```

---

### SQL

```sql
SELECT * FROM blog
WHERE title LIKE '%xxx%'
   OR content LIKE '%xxx%'
ORDER BY create_time DESC;
```

---

### 前端实现

```vue
<input v-model="keyword" placeholder="搜索..." />
<button @click="search">搜索</button>
```

---

## 3.3 随机阅读（重点）

### 后端接口

```http
GET /api/blog/random?size=5
```

---

### SQL（推荐）

```sql
SELECT * FROM blog ORDER BY RAND() LIMIT 5;
```

（数据量大时优化👇）

```sql
SELECT * FROM blog
WHERE id >= (SELECT FLOOR(RAND() * (SELECT MAX(id) FROM blog)))
LIMIT 5;
```

---

### 前端组件

```vue
<div class="random-list">
  <div v-for="item in list" :key="item.id">
    {{ item.title }}
  </div>
</div>
```
