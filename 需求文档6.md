二、功能一：字数统计 + 阅读时间


---

2.1 功能说明

在文章标题下显示：

本文约 1500 字，预计阅读 6 分钟


---

2.2 计算规则（必须统一）

字数统计

中文：1字 = 1字数

英文：按单词或字符（简单可直接 length）


const wordCount = content.length


---

阅读时间

标准：

平均阅读速度：300 字 / 分钟

const minutes = Math.ceil(wordCount / 300)


---

2.3 后端实现（推荐）

👉 在保存文章时计算（避免重复计算）

int wordCount = content.length();
int readTime = (int) Math.ceil(wordCount / 300.0);


---

2.4 数据库字段（新增）

ALTER TABLE blog
ADD COLUMN word_count INT,
ADD COLUMN read_time INT;


---

2.5 前端展示

<div class="meta">
  本文约 {{ blog.wordCount }} 字 · 阅读 {{ blog.readTime }} 分钟
</div>


---

2.6 样式建议

.meta {
  color: #999;
  font-size: 14px;
  margin-bottom: 10px;
}

三、功能二：文章反馈（免登录）


---

3.1 功能说明

在文章底部增加：

👍  ❤️  🎉

特点：

无需登录

每个用户（浏览器）只能点一次

实时显示数量



---

3.2 技术方案

👉 使用 IP + 浏览器标识（localStorage）


---

3.3 数据库设计

CREATE TABLE blog_reaction (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    blog_id BIGINT,
    type VARCHAR(20), -- like / love / celebrate
    count INT DEFAULT 0
);


---

3.4 Redis优化（推荐）

blog:reaction:{blogId}:{type} -> count


---

3.5 接口设计


---

获取反馈

GET /api/blog/reaction/{id}


---

提交反馈

POST /api/blog/reaction

{
  "blogId": 1,
  "type": "like"
}


---

3.6 前端实现

<div class="reaction">
  <span @click="react('like')">👍 {{ data.like }}</span>
  <span @click="react('love')">❤️ {{ data.love }}</span>
  <span @click="react('celebrate')">🎉 {{ data.celebrate }}</span>
</div>


---

3.7 防重复点击（关键）

if (localStorage.getItem("reacted_" + blogId)) return

localStorage.setItem("reacted_" + blogId, true)

四、功能三：毛玻璃导航栏（高级感）


---

4.1 功能说明

滚动时：

背景半透明

模糊（blur）

显示下层背景



---

4.2 CSS实现（核心）

.navbar {
  position: fixed;
  top: 0;
  width: 100%;

  background: rgba(255,255,255,0.6);
  backdrop-filter: blur(10px);

  border-bottom: 1px solid rgba(255,255,255,0.3);
}


---

4.3 滚动增强（推荐）

window.addEventListener("scroll", () => {
  const nav = document.querySelector(".navbar")

  if (window.scrollY > 50) {
    nav.classList.add("scrolled")
  } else {
    nav.classList.remove("scrolled")
  }
})


---

4.4 滚动样式

.navbar.scrolled {
  background: rgba(255,255,255,0.8);
}


---

4.5 暗色模式适配

[data-theme="dark"] .navbar {
  background: rgba(30,30,30,0.6);
}

五、功能四：文章热度地图（重点加分项）


---

5.1 功能说明

类似 GitHub：

👉 展示一年中每天是否发布文章
👉 强化“持续创作”的成就感


---

5.2 数据来源

SELECT DATE(create_time) as date, COUNT(*) as count
FROM blog
WHERE user_id = ?
GROUP BY DATE(create_time);


---

5.3 数据格式（后端返回）

[
  { "date": "2026-03-01", "count": 2 },
  { "date": "2026-03-02", "count": 0 }
]


---
