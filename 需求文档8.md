# 二、功能一：标签系统（Tag System）

---

## 2.1 功能说明

支持：

* 给博客添加多个标签
* 标签分类展示
* 标签云（按热度展示）
* 点击标签筛选文章

---

## 2.2 数据库设计（核心）

---

### 标签表

```sql
CREATE TABLE tag (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) UNIQUE,
    create_time DATETIME
);
```

---

### 博客-标签关系表（多对多）

```sql
CREATE TABLE blog_tag (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    blog_id BIGINT,
    tag_id BIGINT
);
```

---

## 2.3 接口设计

---

### 获取标签列表

```http
GET /api/tag/list
```

---

### 给博客添加标签

```http
POST /api/blog/tag
```

```json
{
  "blogId": 1,
  "tagIds": [1,2,3]
}
```

---

### 根据标签查询博客

```http
GET /api/blog/byTag/{tagId}
```

---

## 2.4 标签云（重点）

---

### 数据统计

```sql
SELECT t.name, COUNT(bt.blog_id) as count
FROM tag t
LEFT JOIN blog_tag bt ON t.id = bt.tag_id
GROUP BY t.id;
```

---

### 前端实现

```vue
<span 
  v-for="tag in tags" 
  :style="{ fontSize: tag.count * 2 + 'px' }"
>
  {{ tag.name }}
</span>
```

---

## 2.5 UI样式建议

```css
.tag {
  background: #FFD1DC;
  padding: 5px 10px;
  border-radius: 20px;
  margin: 5px;
  cursor: pointer;
}
```

# 三、功能二：Markdown 编辑器增强（重点）

---

## 3.1 目标

从“普通文本框”升级为：

👉 **接近 GitHub 的 Markdown 编辑体验**

---

## 3.2 技术选型（强烈推荐）

👉 编辑器：

* **Vditor（首选）**
* 或：

  * mavon-editor（简单）
  * milkdown（高级）

---

## 3.3 必须支持能力

| 功能   | 技术           |
| ---- | ------------ |
| 代码高亮 | highlight.js |
| 数学公式 | KaTeX        |
| 流程图  | mermaid      |
| 实时预览 | 编辑器自带        |

---

## 3.4 示例（Vditor初始化）

```js
import Vditor from 'vditor'

const vditor = new Vditor('editor', {
  height: 500,
  mode: 'sv',
  preview: {
    hljs: { style: 'github' },
    math: true,
    mermaid: true
  }
})
```

# 四、功能三：草稿箱（Draft System）

---

## 4.1 功能说明

支持：

* 自动保存
* 手动保存草稿
* 查看草稿列表
* 发布草稿

---

## 4.2 数据库设计

---

### blog 表新增字段

```sql
ALTER TABLE blog
ADD COLUMN status VARCHAR(20) DEFAULT 'published';
```

---

## 状态定义：

| 状态        | 含义  |
| --------- | --- |
| draft     | 草稿  |
| published | 已发布 |

---

## 4.3 自动保存（核心）

---

### 前端实现

```js
setInterval(() => {
  saveDraft()
}, 5000)
```

---

### 保存接口

```http
POST /api/blog/saveDraft
```

---

## 4.4 草稿列表

```http
GET /api/blog/draftList
```

---

## 4.5 发布草稿

```http
POST /api/blog/publish/{id}
```

---

## 4.6 UI设计

---

### 草稿提示

```vue
<div class="draft-tip">
  已自动保存
</div>
```

---

### 草稿列表页

```vue
<div v-for="item in drafts">
  {{ item.title }}
  <button @click="edit(item.id)">编辑</button>
</div>
```
