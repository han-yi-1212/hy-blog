# 一、项目概述

## 1.1 项目名称

**Blue Sakura Blog（蓝色樱花博客）**

## 1.2 项目目标

开发一个具备以下能力的个人博客系统：

* 用户注册 / 登录
* 发布博客
* 浏览博客
* 点赞功能
* 二次元蓝色风格 UI
* 首页蓝色樱花飘落动画
* 支持自定义背景图片（你后期填充）

---

# 二、技术选型（结合你背景，最优解）

你是做 **Java + AI方向**，所以我给你一个**工程化 + 可扩展方案**：

## 2.1 后端

* **Spring Boot 3.x**
* **Spring Security + JWT（登录认证）**
* **MySQL 8**
* **MyBatis-Plus（提高开发效率）**
* **Redis（缓存点赞 + 登录状态）**

## 2.2 前端

* **Vue 3 + Vite**
* **Pinia（状态管理）**
* **Vue Router**
* **Element Plus（UI组件）**
* **Canvas / CSS 动画（樱花）**

## 2.3 部署

* Docker（推荐）
* Nginx（反向代理）
* 云服务器（阿里云 / 腾讯云）

# 八、蓝色樱花动画实现（核心亮点）

## 8.1 实现方式

使用 Canvas：

```js
const canvas = document.getElementById("sakura");
const ctx = canvas.getContext("2d");

let petals = [];

function createPetal() {
  return {
    x: Math.random() * canvas.width,
    y: Math.random() * canvas.height,
    speedY: Math.random() * 2 + 1,
    speedX: Math.random() * 1 - 0.5,
    size: Math.random() * 5 + 5
  };
}

function drawPetal(p) {
  ctx.fillStyle = "rgba(100,150,255,0.8)"; // 蓝色樱花
  ctx.beginPath();
  ctx.arc(p.x, p.y, p.size, 0, Math.PI * 2);
  ctx.fill();
}

function update() {
  ctx.clearRect(0,0,canvas.width,canvas.height);

  petals.forEach(p => {
    p.y += p.speedY;
    p.x += p.speedX;

    if (p.y > canvas.height) {
      p.y = 0;
    }

    drawPetal(p);
  });

  requestAnimationFrame(update);
}

for (let i = 0; i < 50; i++) {
  petals.push(createPetal());
}

update();
```
